DATA FLOW
================================================================================

User Interaction:
   │
   ├──> PreSelectionUI Component
   │    │
   │    ├─> User selects Wine Tribe
   │    │   └─> Service: getVintagesByTribe(tribe)
   │    │       └─> Returns filtered vintage list
   │    │
   │    ├─> User selects Vintage
   │    │   └─> State updates with selection
   │    │
   │    └─> User clicks "Generate Ticket"
   │        └─> Service: generateDigitalTastingTicket(...)
   │            ├─> Service: generateProvisionalUUID()
   │            ├─> Service: evaluateTQStatus(age, consent)
   │            └─> Returns DigitalTastingTicket object
   │
   ├──> DigitalTastingTicket Component
   │    │
   │    ├─> Display QR code and ticket details
   │    │
   │    └─> User clicks "Validate"
   │        └─> Service: validateQRTicket(qrCode)
   │            ├─> Measures validation time
   │            └─> Returns true/false (<3 sec requirement)
   │
   └──> OperatorDashboard Component
        │
        ├─> Service: calculateOperatorMetrics(pours, minutes, preSelection)
        │   └─> Returns metrics (throughput, avg time, etc.)
        │
        └─> Service: generateTribeBreakdown(tribeArray)
            └─> Returns tribal analytics with percentages

TQStatus (Union Type)
├─ 'GREEN' (Age 19+, FPIC consent)
├─ 'YELLOW' (Age verified, consent pending)
└─ 'RED' (Age <19 or no consent)

VintageOption (Interface)
├─ id: string

├─ description: string
└─ price?: number

OperatorMetrics (Interface)

├─ tqRedCount: number

SERVICE LAYER FUNCTIONS (9 Total)
================================================================================

User Identity:
└─ generateProvisionalUUID() → string
   Returns: Unique provisional user ID for Digital Salon

TQ Safety Gate:
└─ evaluateTQStatus(age, fpicConsent) → TQStatusObject
   Returns: GREEN/YELLOW/RED status with verification details

Ticket Management:
├─ generateDigitalTastingTicket(...) → DigitalTastingTicket
│  Creates QR-based pour authorization ticket
│
└─ validateQRTicket(qrCode) → Promise<boolean>
   Validates ticket in <3 seconds (CEP-23 spec)

Wine Selection:
├─ getVintagesByTribe(tribe) → VintageOption[]
│  Filters wine catalog by tribe selection
│
└─ getWineCatalog() → VintageOption[]
   Returns full wine inventory

Analytics:
├─ calculateOperatorMetrics(...) → OperatorMetrics
│  Computes throughput, avg time, +25% boost
│
├─ generateTribeBreakdown(tribes[]) → TribeBreakdown[]
│  Creates tribal analytics with percentages
│
└─ estimateWaitTime(queueLength) → number
   Calculates estimated wait time in minutes
